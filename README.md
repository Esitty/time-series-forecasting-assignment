## Project Overview
This project focuses on time-series forecasting as part of my MSc Data Analytics coursework.
The objective is to analyze historical data to identify trends, seasonality, and patterns that
support data-driven forecasting and operational planning.

## Objectives
- Explore and clean time-series data
- Identify trends and seasonal patterns
- Compare forecasted values against actual outcomes

## Tools & Techniques
- Python (pandas, numpy, matplotlib)
- Time-series analysis techniques
- Exploratory data analysis

## Key Outcomes
- Identified trend and seasonality patterns in the dataset
- Demonstrated how forecasting supports planning and performance evaluation

## Context
Academic project completed as part of the MSc Data Analytics program.
